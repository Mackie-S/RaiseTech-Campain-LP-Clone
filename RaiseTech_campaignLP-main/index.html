受講生のインタビュー

よくある質問

  • Q.初心者・業界知識がなくても大丈夫？
  • A.基本的なPC操作が出来れば問題ありません。講師や運営、公式メンターが徹底的にサポートします！
  • Q.必要なPC、スペックは？(Macは必要？)
  • A.【最低スペック】 OS：Windows10以降、macOS Mojave（バージョン 10.14）以降 CPU：Intel Core i5以上 メモリ：8GB以上 ストレージ：256GB 【推奨スペック】 OS：macOS Mojave（バージョン 10.14）以降 CPU：Intel Core i7以上、M1 メモリ：16GB以上 ストレージ：512GB(SSD搭載推奨) ※最新のMacBook Proだと推奨スペックをカバーできます。 ※新しく購入するならMacの認定整備済製品も狙い目です。 PCの買い替えは講座受講後に決めても大丈夫です！必ず購入しなければならないわけではございません。
  • Q.どうしてオンライン講座形式なの？
  • A.オンラインでのリアルタイム講座形式は固定の時間が必要で、講師の拘束時間も長いから、基本的にコストパフォーマンスは悪いです。 でも、それは運営する側の都合であって、質疑応答が重要なこうしたスクールでは講師と受講生の距離感がとても大事です。 また常にリアルタイムの情報を渡すには講座形式にする必要があって、現場を意識したRaiseTechならではのスタイルになります。 なので、動画教材も駆使しつつも全国すべての人が受講できるオンライン講座形式に拘っています。
  • Q.学習時間の目安ってどれくらい？
  • A.個人差が出る部分はありますが、おおよそ300時間確保すると、結果を出しやすくなります！ 最速で稼げるようになるには現場で通用しないといけないから、どうしても学習時間は増えます。
  • Q.動画教材やテキスト教材はあるの？
  • A.動画教材は随時更新される形で準備しています。 有料動画教材についてもリリース予定です。テキストについては講座で使う資料があります。
  • Q.講師が月単価80万以上、というのは額面？それとも手取り？
  • A.契約先の顧客と行う契約上の単価を基準としているから、額面、というのがイメージに最も近しいです。 契約単価となるから、正社員でもRaiseTechの講師をしている方はいます。
  • Q.現役エンジニアというのはエンジニア経験がある、という意味？
  • A.現役エンジニアとは今まさに現場に出ているエンジニア、という意味です。 ですので、講師は全員副業という形で講師を行っています。 現場で得た知識や経験・技術をリアルタイムに伝えています。
  • Q.年齢制限はあるの？
  • A.特にありません。どのような方でも受講OKです。
  • Q.受講後、受講中の転職支援で案件の紹介は定期的に行ってくれるの？
  • A.RaiseTechでは自走力をつけて最速で稼げる人材に成長してもらうのが目的です！ 定期的な案件紹介は自走から遠のいてしまいます！ですので、なるべく自身で案件を見つけたり、転職を成功させたりといった、就職・転職支援に力を入れています。
  • Q.質疑応答や録画の閲覧期間はいつまで？
  • A.半永久的に閲覧可能です！一定期間で録画については削除されますが、申請してくれたら録画については最新版を常に閲覧できます。 質疑応答もSlackというコミュニケーションツール内で実施されるんだけど、自身から退会しない限りはいつでも利用可能だよ！
  • Q.質問内容はカリキュラムに沿ったもののみなの？
  • A.いいえ。ありとあらゆる内容に回答します！質問回数に制限はなく、質問内容も不問！なんでも質問してください！ キャリアの相談、転職先の相談、勉強方法、学習手順、課題の質問など一般的なものから、 自分が行おうとしているビジネスプランの添削、戦略に関する相談や、法務・税務などの専門的な内容まで全て対応可能！ 現役エンジニアが対応するので、技術的な質問についてはほぼ全ての内容に対応可能で、代表がビジネスや投資、コンサルタントをしていることもあって、エンジニアリングに無関係の内容でも質疑応答が可能なんだ〜！
  • Q.質問対応の時間はいつからいつまで？
  • A.質問自体は24時間いつでも大丈夫です！迅速に対応します！
  • Q.質問はテキストのみなの？
  • A.必要に応じてZOOMなどのアプリを使ったマンツーマンでの対応もします！ ただし注意点ですが、常にマンツーマンを行うといった形ではないから、必要に応じて要請してください。
  • Q.難易度が高いらしいから、挫折しないか心配…。
  • A.正直に言いますが、挫折率は決して低くありません。ただ、受講生自身が「やる」と決めて行動していただけるのであれば、全力でサポートします！ 完全未経験、業界知識がない比較的高齢の方でも転職に成功した実績があるので、気持ちさえあれば必ずやり遂げられます！ そしてモチベーション維持も含めて講師・運営一同でサポートさせていただきます！
